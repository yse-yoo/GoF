Pattern 解説: Observer

  • 1. 疎結合な通知システム 通知を送る側（Subject）は、受け取る側（Observer）が具体的に誰であるかを知らずに情報を送れます。
  • 2. 動的な購読管理 実行時に subscribe するだけで、新しい通知先を自由に追加・削除できます。

js/observer.js

JavaScript 
// 主体: 監視者のリストを持ち、一斉に叩く
class Subject {
    constructor() { this.observers = []; }
    
    subscribe(obs) { this.observers.push(obs); }
    
    notify(data) {
        this.observers.forEach(obs => obs.update(data));
    }
}

// 監視者: updateメソッドを実装しておく
class UIObserver {
    update(data) { 
        this.element.innerText = `受信: ${data}`; 
    }
}
