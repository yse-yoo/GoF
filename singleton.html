Singleton Pattern

現在のモード: LIGHT

Design Pattern: Singleton

  • 1. 唯一性の確保 new ThemeManager() を何度呼んでも、最初に生成された同一のインスタンスが返されます。
  • 2. グローバルな状態共有 「今のテーマは何か？」という状態を一つのオブジェクトが集中管理するため、UIの不整合が起きません。

js/singleton.js

JavaScript 
class ThemeManager {
    constructor() {
        // インスタンスが既に存在すればそれを返す
        if (ThemeManager.instance) {
            return ThemeManager.instance;
        }
        this.theme = 'light';
        ThemeManager.instance = this;
    }

    toggleTheme() {
        this.theme = this.theme === 'light' ? 'dark' : 'light';
        this.applyTheme();
    }
}
