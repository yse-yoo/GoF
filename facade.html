Facade Pattern

Lights: OFF
Music: OFF
Screen: OFF

Pattern 解説: Facade

  • 1. 複雑さの隠蔽 利用側はサブシステム（照明等）の個別の操作方法を知る必要はなく、Facadeのメソッドを呼ぶだけです。
  • 2. 一括制御（オーケストレーション） 複数の独立したクラスを適切な順番で呼び出す手順を一箇所にまとめ、再利用性を高めます。

js/facade.js

JavaScript 
// ファサード (窓口): 複数のサブシステムを所有する
class HomeCinemaFacade {
    constructor() {
        this.lights = new Lights();
        this.music  = new Music();
        this.screen = new Screen();
    }

    // 複雑な手順をシンプルなメソッドにまとめる
    watchMovie() {
        this.lights.off();
        this.screen.show();
        this.music.play();
    }
}

// 利用側: 面倒な手順はFacadeに丸投げ
const cinema = new HomeCinemaFacade();
cinema.watchMovie();
