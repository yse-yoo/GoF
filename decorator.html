Pattern 解説: Decorator

  • 1. 再帰的なラッピング 「中身」を「装飾」で包み、その装飾をさらに別の装飾で包むことで機能を積み上げます。
  • 2. 組み合わせの爆発を防止 継承を使うと「太字+枠線」などの組み合わせごとにクラスが必要ですが、これは単体部品の合成で済みます。

js/decorator.js

JavaScript 
// デコレーターのベース: 中身(component)を保持する
class TextDecorator {
    constructor(component) { this.component = component; }
    render() { return this.component.render(); }
}

// 具体的な装飾: 既存のrender結果をタグで包む
class BoldDecorator extends TextDecorator {
    render() { 
        return `<strong>${super.render()}</strong>`; 
    }
}

// 実行時:  onion-style でラップしていく
let content = new TextComponent(val);
if (bold) content = new BoldDecorator(content);
